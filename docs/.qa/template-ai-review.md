## Brief

The exact prompt the model received. Recorded verbatim so a future reader can judge what was asked.

## Model output

The model's response, recorded as-is, no edits. If the output is long, link the full transcript from a private
store and quote the verdict paragraph here.

## Human triage

For each item the model raised: short title, triage outcome (`valid`, `false positive`, `out of scope`,
`already fixed`), and a short note on why. If valid, the commit or issue that resolves it.

## Notes for next time

Anything about the brief or the surface that should change before the next review. Kept short.
